App web para saber información de plantas

Este proyecto se enfoca en brindar información detallada sobre diversas plantas a través de una API especializada. Con esta herramienta, los usuarios pueden explorar el nombre científico y común de las plantas, así como descubrir datos esenciales como su ciclo de vida, requerimientos de riego y exposición al sol. La interfaz amigable facilita la búsqueda y el conocimiento.
